# 风格规范.md

## 项目名称

**双生年轮：微博舆论场中《年轮》原唱与版权争议的传播可视分析**

本项目以微博数据为主，辅助平台做对照案例，通过可视化展示张碧晨与汪苏泷围绕《年轮》的原唱、版权与授权争议，揭示社交网络中的传播模式、叙事分化与平台差异。

---

## 一、统一配色规范（方案七 · 赤陶松烟）

本项目采用**赤陶赭石**与**松烟墨绿**的冷暖对照，体现"双生叙事"从共同记忆到自然分裂的过程。配色灵感取自"年轮"的大地属性：赤陶代表时间沉淀的温暖回忆与原唱叙事，松烟代表创作生长的理性力量与版权叙事。

| 回忆情绪 | 回忆 / 遗憾 / 怀旧 / 难过 | `#D4B898` | 回忆杀、遗憾、情感沉淀、平台对比中的"沉淀"维度 |

### 使用要求

- `support_zhang` 永远使用赤陶色 `#C07858`，深变体 `#9A5A40`，浅变体 `#D8B0A0`。
- `support_wang` 永远使用松烟色 `#5A7A6A`，深变体 `#3A5A4A`，浅变体 `#90B0A0`。
- `neutral` / `unclear` 永远使用灰色 `#8C8C8C`。
- `anti_fanwar` / `angry` / 冲突情绪永远使用深绯红 `#A05050`。
- `memory_emotion` / `sad` 永远使用沙金色 `#D4B898`。
- `legal_discussion` / `copyright_authorization` / `confused` 永远使用灰绿 `#6A8A7A`。
- `mocking` 永远使用灰褐 `#B08070`。
- 不允许在不同图中随意更换颜色含义。
- 浅色可以作为填充色，但文字和线条必须保证清晰可读。

### 透明度使用规范

| 场景 | 透明度 | 说明 |
|---|---|---|
| 面积图填充 | `alpha=0.25` | 底层填充，确保折线清晰 |
| 堆叠面积图/流图 | `alpha=0.30~0.35` | 多层叠加时仍需区分各层边界 |
| 传播网络连线（同阵营） | `alpha=0.35` | 细线，大量叠加后高频路径自然浮现 |
| 传播网络连线（跨阵营/混合） | `alpha=0.40~0.45` | 交叉区使用重叠色 `#9A8A7A`，稍高透明度确保可见 |
| 桑基图流动带 | `alpha=0.40~0.50` | 源节点色，重叠时自然混合 |
| 热力图单元格 | `alpha=1.00` | 使用颜色深浅而非透明度表达数值 |
| 节点描边/文字描边 | `alpha=1.00` | 必须完全不透明 |

**避坑**：透明度低于 `0.20` 时元素几乎不可见，高于 `0.70` 时失去半透明混合意义。重叠区颜色（如赤陶 `#C07858` 与松烟 `#5A7A6A` 以 `alpha=0.4` 叠加）会自然呈现灰褐色 `#B8A898`，直观展现"共性话题"，无需手动绘制混合色。
